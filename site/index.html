RL Run Preflight

Catch async GRPO config hazards before you burn GPU hours on prime-rl / verifiers runs.

60-second demo

git clone https://github.com/Abhishek21g/rl-run-preflight.git
cd rl-run-preflight && pip install -e ".[dev]"
rl-preflight run examples/intellect-style-grpo.yaml --mock
rl-preflight run examples/overflow-risk.yaml --mock   # FAIL
rl-preflight doctor out/receipts/latest/summary.json

What it checks

  • Importance-ratio exp(trainer_logprob - inference_logprob) overflow (prime-rl#2972)
  • Masked-token 0 × inf → NaN hazards
  • importance_ratio_max unset vs stale async rollouts
  • Async off-policy lag vs max_off_policy_lag
